Solução 1.5.2.1.
\begin{equation*} R_{n} = \sum_{i=1}^{n} f(x_i) \Delta x . \end{equation*}
Assim \(\Delta x\) é:
\begin{equation*} \Delta x = \frac{b-a}{n} = \frac{4-0}{n} = \frac{4}{n} \end{equation*}
Usando as extremidades à direita de cada subintervalo temos \(x_i = 0 + \Delta x\cdot i = \frac{4 i}{n}\text{.}\) Logo, \(R_n = \displaystyle \sum_{i=1}^{n} f(x_i) \Delta x\)\(= \displaystyle \sum_{i=1}^{n} \left(\frac{4}{n}\right) f\left(\frac{4 i}{n}\right)\)\(= \displaystyle \frac{4}{n}\cdot\sum_{i=1}^{n} \left[\frac{\left(\frac{4 i}{n}\right)^2}{2}\right] + 6\)\(= \displaystyle \frac{4}{n}\cdot\sum_{i=1}^{n} \left(-8\cdot\frac{i^2}{n^2} + 6\right)\)\(= \displaystyle \frac{4}{n}\cdot\left[ \left( \sum_{i=1}^{n} -8\cdot\frac{i^2}{n^2} \right) + 6 n \right]\)\(= \displaystyle \frac{4}{n}\cdot\left[ \left(\frac{-8}{n^2}\right) \cdot \left(\sum_{i=1}^{n} i^2\right) + 6 n \right]\)\(= \frac{4}{n}\cdot\left[ \left(\frac{-8}{n^2}\right) \cdot \left(\frac{n(n+1)(2n+1)}{6}\right) + 6 n \right]\)\(= \frac{4}{n}\cdot\left[ \frac{-8(n+1)(2n+1) + (6)(n)(6)n}{6n} \right]\)\(= \frac{(4)(-8)(n+1)(2n+1) + (4)(6)(6)n^2}{6n^2}\)\(= \frac{(4)(-8)(n+1)(2n+1)}{6n^2} + (4)(6)\)\(= \frac{-5.33333333333333(n+1)(2n+1)}{n^2} + 24\) o que responde a primeira parte do problema acima. Para a segunda parte do problema, vamos tomar o limite com \(n \rightarrow \infty\text{:}\) \(\displaystyle \lim_{n \rightarrow \infty} R_n = \lim_{n \rightarrow \infty} \left[ \frac{-5.33333333333333(n+1)(2n+1)}{n^2} + 24 \right]\)\(\displaystyle = \lim_{n \rightarrow \infty} \left[ \frac{-5.33333333333333(n+1)(2n+1)}{n^2} \right] + 24\)\(\displaystyle = \lim_{n \rightarrow \infty} \left[ \frac{-5.33333333333333(2n^2+3n+1)}{n^2} \right] + 24\)\(\displaystyle = \lim_{n \rightarrow \infty} \left[ -10.6666666666667 + \frac{-16}{n} + \frac{-5.33333333333333}{n^2} \right] + 24\)\(\displaystyle = \lim_{n \rightarrow \infty} \left(-10.6666666666667\right) + \lim_{n \rightarrow \infty} \left(\frac{-16}{n}\right) + \lim_{n \rightarrow \infty} \left(\frac{-5.33333333333333}{n^2}\right) + 24\)\(\displaystyle = -10.6666666666667 + 0 + 0 + 24 = 13.3333333333333\)

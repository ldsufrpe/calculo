Calculando o comprimento de arco.

  1. Divida o intervalo \([a,b]\) em \(n\) subintervalos de mesma largura, cujos extremos de cada subintervalo são \(a=x_1 \lt x_1 \lt x_2\lt \ldots\lt x_n=b.\)

  2. Encontre o comprimento dos segmentos de reta conectando os pontos \(P_i=(x_i,f(x_i))\) e \(P_{i+1}=(x_{i+1},f(x_{i+1}))\) para \(i=0,1\ldots, n-1\text{.}\)

  3. Some todos esses comprimentos para obter uma aproximação do comprimento do arco.

  4. Finalmente, tomamos o limite com \(n\to \infty\) para obter o comprimento de arco.

Exercício 8.
\(\displaystyle\int \tan (8 x + 1) \,dx \,=\,\) \(+ \ C\)
Resposta.
\((-0.125)\ln\!\left(\left|\cos\!\left(8x+1\right)\right|\right)\)
Solução.
Usando o fato de
\begin{equation*} \tan(t) = \dfrac{\sin(t)}{\cos(t)}, \end{equation*}
podemos reescrever a integral como
\begin{equation*} \displaystyle\int \tan (8 x + 1) \,dx = \int \frac {\sin(8 x +1)}{\cos (8 x +1) }\,dx . \end{equation*}
Seja \(u=\cos(8 x +1)\text{,}\) então \(du = -8 \sin(8 x +1)\,dx\text{.}\)
Portanto,
\begin{equation*} \displaystyle\int \tan (8 x + 1) \,dx = \int \frac {du}{-8 u} = -\frac {1}{8} \ln |u| + C = -\frac {1}{8} \ln \left| \cos(8 x + 1) \right| +C \end{equation*}

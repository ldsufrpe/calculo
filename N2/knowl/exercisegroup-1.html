Grupo de exercícios 1.5.4–15.

Use o método de substituição para calcular as seguintes integrais indefinidas.
4.
\(\displaystyle \int x^{5}\left(5+12 x^{6}\right)^{8}\, dx\, =\) + C
Resposta.
\(\frac{\left(5+12x^{6}\right)^{9}}{648}\)
5.
\(\displaystyle \int 6^{4 x} \, dx\,=\,\) \(+ \ C\)
Resposta.
\(\frac{6^{4x}}{7.16704}+C\)
Solução.
Escolha \(u = 4 x\text{.}\) Então \(du = 4 \,dx\text{.}\)
\begin{equation*} \begin{array}{rcl} \displaystyle \int 6^{4 x} \, dx \amp =\amp \displaystyle \dfrac{1}{4} \int 6^u \, du \\ \amp =\amp \dfrac{1}{4} \cdot \dfrac{6^u}{\ln 6} + C\\ \amp =\amp \dfrac{6^{4 x}}{4 \ln 6} + C. \\ \end{array} \end{equation*}
6.
\(\displaystyle \int \frac{7 x^6}{(x^7 + 1)^2} dx \,= \,\) \(+ \ C\)
Resposta.
\(\frac{-1}{x^{7}+1}\)
7.
\(\displaystyle \int \dfrac{2x - 4}{x^2 - 4 x - 21} \, dx \,=\,\)
Resposta.
\(\ln\!\left(x^{2}+\left(-4\right)x+\left(-21\right)\right)+C\)
Solução.
Escolha \(u = x^2 - 4 x - 21\text{.}\) Então \(du = 2x - 4 \,dx\text{.}\)
\begin{equation*} \begin{array}{rcl} \displaystyle \int \dfrac{2x - 4}{x^2 - 4 x - 21} \, dx \amp =\amp \displaystyle \int \dfrac{du}{u} \\ \amp =\amp \ln u + C\\ \amp =\amp \ln|x^2 - 4 x - 21| + C. \end{array} \end{equation*}
8.
\(\displaystyle\int \tan (8 x + 1) \,dx \,=\,\) \(+ \ C\)
Resposta.
\((-0.125)\ln\!\left(\left|\cos\!\left(8x+1\right)\right|\right)\)
Solução.
Usando o fato de
\begin{equation*} \tan(t) = \dfrac{\sin(t)}{\cos(t)}, \end{equation*}
podemos reescrever a integral como
\begin{equation*} \displaystyle\int \tan (8 x + 1) \,dx = \int \frac {\sin(8 x +1)}{\cos (8 x +1) }\,dx . \end{equation*}
Seja \(u=\cos(8 x +1)\text{,}\) então \(du = -8 \sin(8 x +1)\,dx\text{.}\)
Portanto,
\begin{equation*} \displaystyle\int \tan (8 x + 1) \,dx = \int \frac {du}{-8 u} = -\frac {1}{8} \ln |u| + C = -\frac {1}{8} \ln \left| \cos(8 x + 1) \right| +C \end{equation*}

12.
\(\displaystyle \int \frac{-5}{x(\ln x)^2} \, dx \,=\,\)
Resposta.
\(\frac{5}{\ln\!\left(x\right)}+C\)
Solução.
A substituição \(u = \ln x\) nos permite usar a regra da potência para integração para e obter \(\frac{5}{\ln x} + C\text{.}\)

14.
\(\displaystyle \int \frac{x^{5}+1}{\sqrt{x^{6}+6x}} \;dx \,=\,\) \(+ \ C\)
Resposta.
\(\frac{1}{3}\sqrt{x^{6}+6x}\)
Solução.
Para \(u=x^{6}+6x\) temos \({\textstyle\frac{1}{6}} du = (x^{5}+1 )dx\) e portanto;
\begin{equation*} \displaystyle \int \frac{x^{5}+1}{\sqrt{x^{6}+6x}} \;dx = {\textstyle\frac{1}{6}} \int \frac{1}{\sqrt{u}} \;du= {\textstyle\frac{1}{6}}\cdot2\sqrt{u}+C={\textstyle\frac{1}{3}} \sqrt{x^{6}+6x}+C \end{equation*}
15.
\(\displaystyle \int \frac{e^{\sqrt{5y-10}}}{\sqrt{5y-10}} \;dy \,=\,\) \(+ \ C\)
Resposta.
\(0.4e^{\sqrt{5y-10}}\)
Solução.
Para \(u=\sqrt{5y-10}\) temos \(du = \frac{5\, dy}{2 \sqrt{5y-10}}\text{.}\) Logo,
\begin{equation*} \displaystyle \int \frac{e^{\sqrt{5y-10}}}{\sqrt{5y-10}} \;dy = \frac{2}{5} \int e^u \;du= \frac{2}{5} e^{\sqrt{5y-10}}+C \end{equation*}

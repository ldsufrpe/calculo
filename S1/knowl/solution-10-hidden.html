A fórmula da Integral da função potência não é válida para \(x=-1\text{.}\) Logo não é possivel utilizá-la para obter a integral indefinida de \(1/x\text{.}\) O passo natural é encontrar uma função cuja derivada seja \(1/x\text{.}\) Sabe-se que

\begin{equation*} \frac{\dd}{\dd x}\ln{x}= \frac{1}{x}\text{.} \end{equation*}

Então,

\begin{align} \integral{\frac{1}{x}}{x} \amp= \ln{x} + C, \, x \gt 0 \text{.}\tag{\(\star\)} \end{align}

De fato o logaritmo natual não está definido para valores menores que zero. Mas,

\begin{gather} \frac{\dd}{\dd x}\ln{(-x)} = (-1)\frac{1}{-x}=\frac{1}{x} \text{,}\tag{\(\star\star\)} \end{gather}

para \(x \lt 0\text{.}\) Então,

\begin{equation*} \integral{\frac{1}{x}}{x} \ln{(-x)} + C, \quad x \lt 0 \text{.} \end{equation*}

De (\(\star\)) e (\(\star\star\)) concluimos que \(\ln{x}\) é primitiva de \(\sfrac{1}{x}\) se \(x\gt 0\) e que \(\ln{(-x)}\) é primitiva de \(1/x\) se \(x\lt 0\text{.}\) Mais diretamente, podemos afirmar que \(\ln{|x|}\) é uma primitiva de \(1/x\text{.}\) Portanto,

\begin{equation*} \integral{\frac{1}{x}}{x} = \ln{|x|} + C\text{.} \end{equation*}

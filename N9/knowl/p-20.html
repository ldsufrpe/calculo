Parágrafo

Para deduzir a Regra de Simpson, divida o intervalo \([a, b]\) em um número par \(n\) de subintervalos, cada um de largura

\begin{equation*} \Delta x =\frac{b-a}{n}. \end{equation*}

Em \([x_0, x_1\) aproximamos a função \(f\) pelo polinômio de grau dois \(p(x)= Ax^2+ Bx + C\) que passa pelos pontos

\begin{equation*} (x_0, f(x_0), \, (x_1, f(y_1)), (x_2, f(x_2)), \end{equation*}

no qual

\begin{gather*} f(x_0)= p(x_0) = Ax_0^2 + Bx_0 + C \\ f(x_1) = p(x_1) = Ax_1^2 + Bx_1 + C \\ f(x_2) = p(x_2) = Ax_2^2 + Bx_2 + C \end{gather*}

Além disso,

\begin{equation*} x_2 - x_0 = 2\Delta x \end{equation*}

e

\begin{equation*} x_2 + x_0 = 2x_1, \end{equation*}

já que \(x_1=\frac{x_2+x_0}{2}\text{.}\) Pelo Teorema Fundamenta do Cálculo

\begin{align*} \integrald{f(x)}{x_0}{x_2}{x}\amp \approx \integrald{p(x)}{x_0}{x_2}{x} \\ \amp = \left(\frac{A}{3}x^3 + \frac{B}{2}x^2+ Cx\right) \biggr\rvert_{x_0}^{x_2} \\ \amp = \frac{A}{3}(x_2^3-x_0^3) + \frac{B}{2}(x_2^2-x_0^2) + C (x_2-x_0)\\ \amp= \frac{x_2-x_0}{6}\left(2A(x_2^2+ x_2x_0+ x_0^2) + 3B(x_2+x_0) + 6C\right) \\ \amp = \frac{\Delta x}{3}\left((Ax_2^2 + Bx_2^2+ C) + (Ax_0^2 + Bx_0^2+ C)+ A(x_2^2+ 2x_2x_0+x_0^2 + 2B(x_2+x_0) + 4C\right)\\ \amp = \frac{\Delta x}{3}\left(f(x_2) + f(x_0) + A(2x_1)^2 + 2B(2x_1) + 4C\right)\\ \amp = \frac{\Delta x}{3}\left( f(x_0) + 4f(x_1) + f(x_2)\right) \end{align*}

A repetição desse processo nos subintervalos \([x_{i-2}, x_i]\) fornece

\begin{align*} \integrald{f(x)}{a}{b}{x} \amp \approx \frac{\Delta x}{3}\left[f(x_0) + 4f(x_1)+ 2f(x_2) + 4f(x_3) + 2f(x_4) + \ldots + 4f(x_{n-1})+f(x_n)\right] . \end{align*}

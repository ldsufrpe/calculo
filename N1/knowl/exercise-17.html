Exercício 10.

Encontre a função \(f(x)\) sabendo que
\begin{equation*} f''(x) = 8 \sin x, \quad f'(\pi) = 5, \quad f(\pi) = 0 \end{equation*}
\(f(x) =\)
Resposta.
\(-8\sin\!\left(x\right)+\left(-3\right)x+0-\left(-3\right)\pi \)
Solução.
Note que
\begin{equation*} f'(x) = \int 8 \sin x \, dx = -8 \cos x + C_1. \end{equation*}
Encontre \(C_1\) usando \(f'(\pi) = 5\text{.}\)
\begin{equation*} \begin{array}{rcl} -8 \cos \pi + C_1 \amp =\amp 5\\ 8 + C_1 \amp =\amp 5\\ C_1 \amp =\amp -3 \end{array} \end{equation*}
\begin{equation*} f(x) = \int \left( -8 \cos x - 3 \right) \, dx = -8 \sin x - 3 x + C_2. \end{equation*}
Encontre \(C_2\) usando \(f(\pi) = 0\text{.}\)
\begin{equation*} \begin{array}{rcl} -8 \sin \pi - 3 \pi + C_2 \amp =\amp 0\\ C_2 \amp =\amp 0 + 3 \pi \end{array} \end{equation*}
Logo, \(f(x) = -8 \sin x - 3 x + 0 + 3 \pi\text{.}\)

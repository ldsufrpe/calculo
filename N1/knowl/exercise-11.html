Exercício 4.

Considere a função \(\displaystyle f(x) = x^5 + 2 \sqrt{x}\text{.}\)
Seja \(F(x)\) a primitiva de \(f(x)\) com \(F(1) = -6\text{.}\)
Então \(F(x) =\)
Resposta.
\(\frac{x^{6}}{6}+\frac{4x^{1.5}}{3}-7.5\)
Solução.
\(\displaystyle f(x) = x^5 + 2 \sqrt{x} = x^5 + 2 x^{1/2}\text{:}\) \(\displaystyle F(x) = \frac{x^6}{6} + \frac{2 x^{3/2}}{3/2} + c = \frac{x^6}{6} + \frac{4 x^{3/2}}{3} + c\) Já que \(F(1) = -6\text{,}\) temos \(\displaystyle \frac{1}{6} + \frac{4}{3} + c = -6\) \(\displaystyle c = -6 - \frac{1}{6} - \frac{4}{3} = \frac{-135}{18} = -7.5\) Portanto \(\displaystyle F(x) = \frac{x^6}{6} + \frac{4 x^{3/2}}{3} - 7.5\)

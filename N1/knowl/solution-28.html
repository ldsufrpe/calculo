Solução 1.5.8.1.

Para resolver este problema, você precisa lembrar que a derivada de \(\rm{tg(t)}\) é \(\sec^2(t)\text{.}\) Não é difícil derivar isso usando a identidade \(\rm{tg}(t) = \frac{\rm{sen(t)}}{\cos(t)}\) e a regra de cocientes para derivadas.

Com isso em mente, encontre a primitiva geral de \(f(t) = 9\sec^2(t)-{4}t^2\) que será \(F(t) = 9\rm{tg(t)} - \frac{ {4}t^3 }{ 3} + C\text{.}\) Agora usando o fato de \(F(0) = 0\) e que \(\rm{tg}(0) = 0\text{,}\) devemos ter \(C = 0\text{.}\)

Portanto, pondo 1.5 nesta expressão, obtemos:

\begin{equation*} F(1.5) = 9\rm{tg}(1.5) - \frac{ 4(1.5)^3 }{ 3 } = 122.413 \end{equation*}

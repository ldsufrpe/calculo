\begin{equation*} \int \frac{du}{\sqrt{a^2-u^2}} = \sin^{-1} \frac{u}{a} + C \end{equation*}
No nosso caso, \(a = 1\) e \(u = x\text{.}\) Portanto, nossa primitiva é:
\begin{equation*} F(x) = \left(\sin^{-1} x \right) + C \end{equation*}
Então, aplicando o Teorema Fundamental do Cálculo e usando uma calculadora para calcular, obtemos:
\begin{equation*} \begin{aligned} \int_0^{0.7} \frac{dx}{\sqrt{1-x^2}} dx \amp = F(0.7) - F(0) \\ \amp = \left(\sin^{-1} 0.7\right) + C - \left(\sin^{-1} 0\right) - C\\ \amp = 0.7754 \end{aligned} \end{equation*}
